JavaScript Summary

JS Introduction

JavaScript is a programming language for creating interactive and dynamic web content.

JS Where To

JavaScript can be placed inside HTML using the <script> tag or linked as external files with the .js extension.

JS Output

Methods for outputting data include:

  • document.write()
  • console.log()
  • alert()
  • Modifying HTML elements using innerHTML.
JS Statements

Statements are JavaScript instructions executed by the browser, such as variable declarations or function calls.

JS Syntax

JavaScript is case-sensitive. Code blocks are defined with curly braces {}, and statements end with a semicolon ;.

JS Comments

Comments are used to describe code:

  • Single-line: //
  • Multi-line: /* */
JS Variables

Variables are used to store data. Declared using var, let, or const.

JS Let

The let keyword declares block-scoped variables.

JS Const

The const keyword declares variables with values that cannot be changed.

JS Operators

Operators are used to perform operations on variables and values:

  • Arithmetic: +, -, *, /
  • Logical: &&, ||, !
JS Arithmetic

Arithmetic operators include:

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
JS Assignment

Assignment operators, like =, are used to assign values to variables.

JS Data Types

JavaScript data types include:

  • Primitive: String, Number, Boolean, Null, Undefined
  • Object types: Array, Function, Object
JS Functions

Functions are blocks of code designed to perform a specific task:

function greet(name) { return "Hello " + name; }
JS Objects

Objects are collections of properties and values:

let car = { brand: "Toyota", model: "Corolla", year: 2022 };

JavaScript Summary

JS Object Properties

Properti objek adalah atribut yang menyimpan nilai. Contoh:

const car = { brand: "Toyota", year: 2020 };
console.log(car.brand); // Output: Toyota

JS Object Methods

Metode adalah fungsi yang terkait dengan objek. Contoh:

const person = {
    name: "Alice",
    greet: function() {
        return Hello, ${this.name};
    }
};
console.log(person.greet()); // Output: Hello, Alice

JS Object Display

Objek dapat ditampilkan menggunakan console.log() atau JSON.stringify().

const obj = { key: "value" };
console.log(JSON.stringify(obj)); // Output: {"key":"value"}

JS Object Constructors

Konstruktor adalah fungsi untuk membuat objek baru.

function Car(brand, year) {
    this.brand = brand;
    this.year = year;
}
const car = new Car("Toyota", 2020);
console.log(car.brand); // Output: Toyota

JS Events

Event adalah aksi yang dapat diproses oleh JavaScript, seperti klik tombol.

document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});

JS Strings

String digunakan untuk menyimpan teks.

let greeting = "Hello, World!";
console.log(greeting.length); // Output: 13

JS String Methods

Metode string digunakan untuk memanipulasi teks:

  • toUpperCase(): Mengubah menjadi huruf kapital.
  • toLowerCase(): Mengubah menjadi huruf kecil.
  • trim(): Menghapus spasi di awal/akhir string.

JS String Search

Mencari teks dalam string:

let text = "Hello, JavaScript!";
console.log(text.includes("JavaScript")); // Output: true

JS String Templates

Template literal mempermudah penggabungan string:

let name = "Alice";
console.log(Hello, ${name}!); // Output: Hello, Alice!

JS Numbers

Number adalah tipe data untuk angka.

let x = 10, y = 20;
console.log(x + y); // Output: 30

JS BigInt

BigInt digunakan untuk angka yang sangat besar.

let big = 123456789012345678901234567890n;
console.log(big + 1n); // Output: 123456789012345678901234567891n

JS Number Methods

Metode untuk angka:

  • toFixed(): Membulatkan angka ke jumlah desimal tertentu.
  • toString(): Mengubah angka menjadi string.
  • parseInt(): Mengonversi string menjadi bilangan bulat.

JS Number Properties

Properti angka:

  • MAX_VALUE: Nilai maksimal angka.
  • MIN_VALUE: Nilai minimal angka.
  • NaN: Not a Number.

JS Arrays

Array digunakan untuk menyimpan beberapa nilai dalam satu variabel.

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[1]); // Output: Banana

JS Array Methods

Metode untuk array:

  • push(): Menambahkan elemen di akhir array.
  • pop(): Menghapus elemen terakhir.
  • shift(): Menghapus elemen pertama.
  • unshift(): Menambahkan elemen di awal array.

JS Arrays Search

Fungsi Pencarian Array: Gunakan metode seperti indexOf(), lastIndexOf(), atau includes() untuk mencari elemen dalam array.

Pencarian Berdasarkan Kondisi: Gunakan find() untuk menemukan elemen pertama yang memenuhi kondisi atau findIndex() untuk mendapatkan indeks elemen tersebut.

const fruits = ["apple", "banana", "cherry"];
console.log(fruits.indexOf("banana")); // Output: 1
console.log(fruits.includes("cherry")); // Output: true
const numbers = [10, 20, 30];
console.log(numbers.find(num => num > 15)); // Output: 20
JS Arrays Sort

Sorting Dasar: Gunakan sort() untuk mengurutkan elemen array secara alfabetis.

Sorting Numerik: Gunakan fungsi pembanding (callback) seperti (a, b) => a - b untuk sorting angka.

const fruits = ["banana", "apple", "cherry"];
console.log(fruits.sort()); // Output: ["apple", "banana", "cherry"]
const numbers = [4, 2, 5, 1, 3];
console.log(numbers.sort((a, b) => a - b)); // Output: [1, 2, 3, 4, 5]
JS Arrays Iteration

Metode seperti forEach(), map(), filter(), dan reduce() digunakan untuk melakukan iterasi atau operasi pada setiap elemen array.

const numbers = [1, 2, 3];
numbers.forEach(num => console.log(num)); // Logs: 1, 2, 3
const doubled = numbers.map(num => num * 2);
console.log(doubled); // Output: [2, 4, 6]
JS Arrays Const

Array yang dideklarasikan menggunakan const tetap dapat diubah isinya (menambah, menghapus elemen) tetapi tidak dapat mengganti referensinya.

const fruits = ["apple", "banana"];
fruits.push("cherry");
console.log(fruits); // Output: ["apple", "banana", "cherry"]
JS Dates

Objek Date digunakan untuk menangani tanggal dan waktu, misalnya new Date() untuk mendapatkan tanggal/waktu saat ini.

const now = new Date();
console.log(now); // Output: Current date and time
JS Date Formats

Tanggal dapat diformat dengan berbagai cara seperti toDateString(), toISOString(), atau toLocaleDateString().

const now = new Date();
console.log(now.toDateString()); // Output: e.g., "Sun Nov 24 2024"
JS Date Get Methods

Metode seperti getFullYear(), getMonth(), getDate(), dan getDay() digunakan untuk mendapatkan bagian-bagian spesifik dari tanggal.

const now = new Date();
console.log(now.getFullYear()); // Output: 2024
console.log(now.getMonth()); // Output: 10 (November is 0-based)
JS Date Set Methods

Metode seperti setFullYear(), setMonth(), dan setDate() digunakan untuk mengatur nilai tanggal tertentu.

const now = new Date();
now.setFullYear(2030);
console.log(now.getFullYear()); // Output: 2030
JS Math

Objek Math menyediakan fungsi matematika seperti Math.sqrt(), Math.abs(), Math.pow(), dan konstanta seperti Math.PI.

console.log(Math.sqrt(16)); // Output: 4
console.log(Math.PI); // Output: 3.141592653589793
JavaScript Topics
JS Random

Gunakan Math.random() untuk menghasilkan angka acak antara 0 dan 1.

console.log(Math.random()); // Output: Random number between 0 and 1
JS Booleans

Tipe data boolean hanya memiliki dua nilai: true dan false. Digunakan dalam pernyataan logika.

const isSunny = true;
const isRaining = false;

console.log(isSunny);  // Output: true
console.log(isRaining);  // Output: false
JS Comparisons

Operator perbandingan seperti ==, ===, !=, <, > digunakan untuk membandingkan nilai.

console.log(5 == "5");  // Output: true (loose comparison)
console.log(5 === "5");  // Output: false (strict comparison)
console.log(10 > 5);  // Output: true
console.log(10 <= 5);  // Output: false
JS If Else

Pengambilan keputusan: Gunakan if dan else untuk menjalankan kode berdasarkan kondisi.

const number = 10;

if (number > 5) {
  console.log("Greater than 5");
} else {
  console.log("Less than or equal to 5");
}
JS Switch

Alternatif untuk if-else: Gunakan switch untuk menangani banyak kondisi.

const fruit = "apple";

switch (fruit) {
  case "banana":
    console.log("It's a banana.");
    break;
  case "apple":
    console.log("It's an apple.");
    break;
  default:
    console.log("Unknown fruit.");
}
JS Loops (For)

Gunakan for loop untuk iterasi melalui array atau data lainnya.

const numbers = [1, 2, 3, 4, 5];

for (let i = 0; i < numbers.length; i++) {
  console.log(numbers[i]);
}
